### 내용
특정 성향을 띄는 workload를 사용하는 프로그램을 설정해 성능을 측정

#### Good Benchmarks
- Target Domain이 잘 설정되어야 함
- 어떤 벤치마크를 사용하다보면 그 벤치마크가 필요없어지는 시점이 생김
##### SPEC (Standard Performance Evaluation Corporation)
- CPU 성능에 초점
- I/O 연산이 거의 없음
##### TPC
- OLTP (On-Line Transaction Processing) 성능에 초점
- 데이터 베이스 성능 측정에 활용
- Transaction 성능을 측정
